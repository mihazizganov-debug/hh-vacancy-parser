# 📊 HH Vacancy Parser

**Курсовая работа по работе с базами данных и API**

Программа для сбора и анализа вакансий с сайта hh.ru. Загружает данные о 10 крупных IT-компаниях, сохраняет их в PostgreSQL и предоставляет удобный интерфейс для анализа.

---

## 🚀 Функциональность

- 🔍 **Поиск вакансий** — получает данные о вакансиях 10 компаний через публичное API hh.ru
- 💾 **Сохранение в БД** — автоматически создаёт базу данных и таблицы в PostgreSQL
- 📊 **Анализ данных** — через класс `DBManager` доступны:
  - Список компаний с количеством вакансий
  - Все вакансии с названиями, зарплатами и ссылками
  - Средняя зарплата по вакансиям
  - Вакансии с зарплатой выше средней
  - Поиск вакансий по ключевым словам
- 🖥️ **Консольный интерфейс** — простое меню для взаимодействия

---

## 🛠️ Технологии

- **Python 3.10+**
- **requests** — работа с API hh.ru
- **psycopg2** — работа с PostgreSQL
- **python-dotenv** — хранение конфиденциальных данных
- **Poetry** — управление зависимостями

## 📦 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/mhazizganov-debug/hh-vacancy-parser.git
cd hh-vacancy-parser
```

### 2. Установить Poetry и зависимости

# Установка poetry (если не установлен)
```
pip install poetry
```

# Установка зависимостей
```
poetry install
```

# Активация окружения
```
poetry shell
```

### 3. Настроить подключение к PostgreSQL
Создайте файл .env в корне проекта:
```env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=hh_vacancy_parser
DB_USER=postgres
DB_PASSWORD=ваш_пароль
```

### 4. Запустить программу
```
python main.py
```
При первом запуске программа автоматически:

Создаст базу данных hh_vacancy_parser

Создаст таблицы companies и vacancies

Загрузит данные о компаниях и их вакансиях через API hh.ru

После запуска откроется меню:
